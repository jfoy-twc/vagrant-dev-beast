Vagrant development box for web developers
==========================================

Requirements:
* VirtualBox 
* Vagrant (for Windows 1.1+)
* PuppetLabs Puppet

Currently the base box is 32-bit Ubuntu 12.04.

Pre-installed are:
* PHP 5.4 
* Oracle Java 7 
* NodeJS [TODO]
* Nginx 1.3
* MySQL [TODO]
* Python
* Jetty 9 [TODO]
